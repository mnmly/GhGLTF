## glTF export component for Grasshopper

Grasshopper Component for exporting to glTF

### Installation

```
$ /Applications/RhinoWIP.app/Contents/Resources/bin/yak install ghGLTF # Replace RhinoWIP.app with Rhinocerous.app
```
